# 🌐 ياسين عبدالعال - البورتفوليو الشخصي

مرحبًا بك في البورتفوليو الخاص بي!
المشروع معمول باستخدام **Next.js** و **React** لعرض مهاراتي، خبراتي، وأبرز المشاريع اللي اشتغلت عليها كمطور **واجهات أمامية (Front-End Developer)**.

---

## 🚀 عن المشروع

البورتفوليو بيعرض:

* تصميم عصري، نظيف ومتجاوب مع كل الأجهزة.
* مجموعة من المشاريع الاحترافية والـ Side Projects.
* أداء محسن وSEO باستخدام Next.js.
* أنيميشنز سلسة وعناصر واجهة مستخدم تفاعلية.

---

## 🛠️ التقنيات المستخدمة

* **Next.js** – فريمورك مبني على React بيدعم الـ SSR وتحسين محركات البحث.
* **React** – لتطوير الواجهات بشكل Component-Based.
* **HTML5 / CSS3 / JavaScript** – أساسيات تطوير الويب.
* **Tailwind CSS** – مكتبة Utility-First لتنسيق سريع ومرن.
* مكتبات إضافية ممكن استخدامها حسب الحاجة: **Swiper, Recharts, Firebase, Bootstrap**.

## 🖥️ كيفية تشغيل المشروع محليًا

```bash
# استنسخ المستودع
git clone https://github.com/your-username/Yasin-Marei-s-Portfolio.git

# ادخل على مجلد المشروع
cd Yasin-Marei-s-Portfolio

# ثبّت الاعتمادات
npm install

# شغل السيرفر المحلي
npm run dev
```

ثم افتح [http://localhost:3000](http://localhost:3000) من المتصفح.
