# 🌐 Yassin Abdelaal - Portfolio

Welcome to my personal portfolio!  
This project is built with **Next.js** and **React** to showcase my skills, experience, and selected projects as a **Front-End Developer**.  

---

## 🚀 About the Project
My portfolio highlights:
- A clean, responsive, and modern design.
- A collection of professional and side projects.
- Optimized performance and SEO using Next.js.
- Smooth animations and interactive UI components.

---

## 🛠️ Tech Stack
- **Next.js** – Framework for React with SSR & SEO optimization.
- **React** – Component-based UI development.
- **HTML5 / CSS3 / JavaScript** – Core web technologies.
- **Tailwind CSS** – Utility-first styling.

## 🖥️ Getting Started
To run the portfolio locally:

```bash
# Clone the repo
git clone https://github.com/your-username/Yasin-Marei-s-Portfolio.git

# Navigate to the folder
cd Yasin-Marei-s-Portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
```
